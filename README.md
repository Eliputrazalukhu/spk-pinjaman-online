## Setup Instructions 🚀
1. **Download the source code**: Click on the "Code" button above and select "Download ZIP".
2. **Extract the ZIP file**: Copy the "Loan_Management_System" folder.
3. **Paste the folder inside the XAMPP root directory**: Typically located in `C:\xampp\htdocs`.
4. **Create a database**: Open PHPMyAdmin (http://localhost/phpmyadmin) and create a database named `loan_db`.
7. **Import SQL file**: In the "SQL file" folder of the extracted zip, find and import `loan_db.sql` into the `loan_db` database.
6. **Run the application**: Navigate to http://localhost/Loan_Management_System in your web browser.
